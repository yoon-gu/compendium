현대 금융 시스템은 풍부한 경제적 신호를 담고 있는 대규모 거래(transactional) 데이터와 이벤트 수준(event-level) 데이터를 생성한다.
이 논문은 멀티 소스 뱅킹 이벤트 시퀀스(multi-source banking event sequences)를 위한 파운데이션 모델(foundation model) 계열인 PRAGMA를 제시한다.
저자들은 금융 레코드의 이산적이고 가변 길이인 성격에 맞춘 자기지도(self-supervised) 목적함수와 함께, 마스킹 모델링(masked modelling)으로 트랜스포머(Transformer) 기반 아키텍처를 대규모 이질적 뱅킹 이벤트 코퍼스 위에서 사전학습(pre-training)한다.
이렇게 얻어진 모델은 신용 평가(credit scoring), 사기 탐지(fraud detection), 라이프타임 밸류(lifetime value) 예측 같은 다양한 다운스트림 과제(downstream task)를 지원한다. 추출된 임베딩(embedding) 위에 단순한 선형 모델만 학습해도 강한 성능을 얻을 수 있고, 가벼운 파인튜닝(fine-tuning)을 더하면 성능은 더 올라간다.
폭넓은 다운스트림 평가를 통해, 저자들은 PRAGMA가 원시 이벤트 시퀀스로부터 여러 도메인에서 우수한 성능을 보이며 금융 응용을 위한 범용 표현 계층(representation layer)을 제공함을 보인다.\\

\textbf{고지}: 절대 지표는 상업적으로 민감하므로, 본 논문은 상대 개선폭만 보고한다.\\
모든 예시는 합성 데이터이며 실제 운영 데이터가 아니다.
