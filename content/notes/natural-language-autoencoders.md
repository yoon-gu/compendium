## 한 문장 요약

NLA(Natural Language Autoencoder)는 *자연어를 병목 표현으로 삼는 오토인코더*다 — LLM의 활성화를 자연어 설명으로 옮기는 **활성화 언어화기(AV)** 와, 그 설명만으로 활성화를 복원하는 **활성화 재구성기(AR)** 를 RL로 함께 학습시킨다. 해석 가능성을 명시적으로 강제하지 않았는데도 사람이 읽을 수 있는 설명이 자연스레 등장한다는 점이 핵심.

## 핵심 아이디어

1. 두 모듈 구조 (AV ⇄ AR): 둘 다 대상 LLM의 복사본으로 초기화. AV는 활성화 → 자연어 설명, AR는 자연어 설명 → 활성화 공간 매핑.
2. 유일한 학습 신호 = 재구성 손실: 해석 가능성·충실성 같은 보조 손실 없음. 그럼에도 학습이 진행될수록 설명의 정보량이 단조 증가.
3. 자연어 병목의 두 가지 효과:
   - 비지도성: 라벨이나 사전이 필요 없음 (vs. SAE는 학습된 사전, 지도 미세조정은 라벨 필요).
   - 읽히는 출력: Logit Lens나 SAE처럼 어휘/사전에 묶이지 않음. 사람이 읽고 그대로 인용할 수 있는 형태.
4. 인과적 개입 가능: 설명을 편집해 AR로 원본/편집본을 복원하고 그 차벡터를 모델에 주입 → 모델의 내부 인식을 *말로 조작*할 수 있음.

## 사례 연구가 보여준 것

- 초기 Opus 4.6이 영어 프롬프트에 갑자기 러시아어로 답하는 사례에서, NLA 설명은 모델이 *"사용자가 영어 비원어민이고 모국어가 러시아어"* 라는 가설에 고착되어 있다고 보고. 이를 단서로 학습 데이터에서 *번역된 영어 프롬프트 + 미번역 외국어 응답* 의 잘못된 짝이 발견됨 — 해석가능성 도구가 데이터 버그 디버깅에 직접 기여.
- Claude Haiku 3.5, Haiku 4.5, Opus 4.6 세 모델 모두에서 일관된 정보량 증가 트렌드.
- 사용자에 관해 컨텍스트로부터 추론 가능한 특징(성별, 직업 등)을 표면화.
- 토이 세팅에서, NLA 차벡터로 "어떤 응답이 보상받을지에 대한 모델의 내부 인식" 을 바꿔 행동을 변화시킴.

## 기존 방법과의 차이

| 접근 | 비지도? | 자유 자연어 출력? | 한계 |
|---|---|---|---|
| Logit Lens | ✓ | △ (어휘 토큰에 묶임) | 표현이 어휘에 한정 |
| 희소 오토인코더(SAE) | ✓ | ✗ (학습된 사전 ID) | 특성 라벨링 단계가 또 필요, 커버리지 공백 |
| 활성화 오라클(AO, 지도 미세조정) | ✗ | ✓ | 참값 라벨 필요, 분포가 좁아짐 |
| NLA (본 연구) | ✓ | ✓ | 명시적 정합성 제약 없음 → 검증 도구 필요 |

## 내가 흥미롭게 본 지점

- 단일 RL 목적함수만으로 *해석 가능성이 부수효과로 떠오른다*는 지점 — 자연어 병목 자체가 강한 inductive bias로 작용한다는 가설을 강하게 뒷받침.
- "텍스트로부터 활성화로 매핑하는 모델(AR)" 이 따로 존재한다는 발상은 HyperSteer 같은 선례를 일반화한 것이지만, 사이클 일관성 학습으로 *자연어가 1급 시민(first-class citizen)으로 활성화 공간에 다리를 놓는다*는 그림이 깔끔하다.
- 같은 시기 Chalnev의 *Cycle-Consistent Activation Oracles*도 매우 유사한 결론에 독립적으로 도달 — 두 그룹이 동시기에 같은 방향으로 수렴한다는 사실 자체가 흥미.
